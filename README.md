# 🌟 Atria

Atria é uma plataforma web desenvolvida como Trabalho de Conclusão de Curso (TCC) da Escola Técnica Estadual Bento Quirino, com o objetivo de auxiliar no aprendizado de idiomas por meio da utilização de flashcards e exercícios interativos.

## 📚 Sobre o Projeto

A plataforma permite a importação de decks de flashcards criados em comunidades de aprendizado de idiomas, como o Anki, transformando esse conteúdo em atividades de estudo.

O sistema possui dois perfis principais:

### 👨‍🎓 Aluno (Usuário Comum) 

- Participar de turmas através de um código fornecido pelo professor;
- Acessar os decks disponibilizados pela turma;
- Realizar exercícios gerados a partir dos flashcards;
- Acompanhar seu progresso nos estudos.

### 👩‍🏫 Professor

- Criar turmas;
- Gerar códigos de acesso para os alunos;
- Associar decks de flashcards às turmas;
- Visualizar estatísticas e desempenho dos alunos;
- Gerenciar os participantes da turma.

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/stefanynovais/TelasAtriaPW.git
```

Entre na pasta do projeto:

```bash
cd TelasAtriaPW
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O sistema estará disponível em:

```bash
http://localhost:5173
```
